# Lexico
Lightweight text manipulation scripting language. It acts as a simple, versatile, and expressive evolution of regex for multi-line patterns, supporting:
* Finding
* Replacing
* Logging
* Highlighting

## Graphical User Interface
The main interface for the interpreter is the GUI, written with [imgui](##licensing).
